# Practica ADS - Grupo DT

Repositorio de trabajo del Grupo DT para la practica de Administracion de Sistemas.

## Carpetas principales

- `grupo-dt-CloudFormation/`: solucion final ordenada para la entrega. Incluye CloudFormation, scripts de peering distribuido y codigo de aplicacion usado como referencia.
- `grupo-dt-devops/`: propuesta DevOps con Ansible/Jenkins/GitHub Actions. Se conserva como ampliacion, pero no fue el flujo ejecutado finalmente.
- `docs/`: documentacion auxiliar.

## Documentacion de apoyo

- `PREPARACION_DEFENSA_PRACTICA_ADS.md`: resumen para preparar la defensa.
- `QUE_ANADIR_A_PRACTICAADS_CDE_Y_GRUPO.md`: guia auxiliar usada para completar la memoria.

## Nota de seguridad

No subir credenciales al repositorio. La raiz contiene `.gitignore` para excluir:

- `*.pem`
- `*.key`
- `.env`
